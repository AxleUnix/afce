Цикл с параметром, или цикл со счётчиком, или арифметический цикл - это цикл с заранее известным числом повторов. В блоке модификации указывается закон изменения переменной параметра.

Xo - начальное значение параметра;
h - шаг;
Xn - последнее значение параметра.
Для создания циклов с параметром необходимо использовать правила:
1. Параметр цикла, его начальное и конечное значения, а также шаг должны быть одного типа;
2. Запрещено изменять в теле цикла начальное, текущее или конечное значение для параметра;
3. Запрещено входить в цикл минуя блок модификации;
4. Если начальное значение больше конечного, то шаг - число отрицательное;
5. После выхода из цикла значение переменной параметра неопределённое и не может использоваться в дальнейших вычислениях;
6. Из цикла можно выйти не закончив его, тогда переменная параметра сохраняет своё последнее значение.
